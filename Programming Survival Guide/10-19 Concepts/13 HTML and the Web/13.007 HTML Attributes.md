# id

# target
Controls how a page is opened.
`target="_blank"`
### \_blank
Opens the page in a new window or tab.
### \_self (default)
Opens the page in the same frame as it was clicked.
### \_parent
Opens the page in the parent frame
### \_top
Opens the page in the full body of the window.
